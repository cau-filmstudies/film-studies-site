중앙대학교 영화학과 콘텐츠 관리

콘텐츠 관리 방법:
GitHub 저장소를 통해 콘텐츠를 직접 편집할 수 있습니다.
📁 GitHub 저장소 열기 📝 콘텐츠 폴더 보기
콘텐츠 편집 단계:
  1. GitHub 저장소에 로그인합니다
  2. 편집할 파일을 클릭합니다
  3. 연필 아이콘(✏️)을 클릭하여 편집 모드로 전환합니다
  4. 내용을 수정합니다
  5. 페이지 하단의 "Commit changes" 버튼을 클릭합니다
  6. 변경사항이 자동으로 Netlify에 배포됩니다
주의사항:
• 파일명과 경로는 변경하지 마세요
• Markdown 형식을 유지해주세요
• 이미지는 public/images 폴더에 업로드하세요
주요 콘텐츠 파일:
• content/board/ - 게시판 글
• content/pages/ - 정적 페이지
• public/images/ - 이미지 파일
